# First & Last Letter Extractor
This program asks the user for a word and prints the first and last letter using simple string indexing.

## Features
- Asks the user to type a word
- Extracts the first letter ([0])
- Extracts the last letter ([-1])
- Prints both letters in a formatted output


# Learning Purpose
A small exercise to practice:
- user input
- string indexing
- basic variable usage
- simple formatted printing
